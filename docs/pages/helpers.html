Helpers

To use helper just add a gy-h="helper-name-here" data attribute to the grid/column element.

See the examples below.

No gutter

Remove the gutter from the grid by adding g-no-gutter.


Source Codepen

Horizontal alignment

Apply horizontal alignment by adding g-left, g-center, g-right.

Source Codepen

Horizontal distribution

Apply horizontal distribution by adding g-space-around or g-space-between.


Source Codepen

Vertical alignment, grid level

Apply vertical alignment by adding g-top, g-middle, g-bottom.

Vertical alignment, column level
