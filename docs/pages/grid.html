Grid

Gridy is easy to use. Just add a gy="g" data attribute to make the grid and a gy="c" data attribute to make the columns.

Let's see some examples.

Automatic width

Add gy="g" to the parent and gy="c" to the children, and the columns will get automagically equal width size.

Source Codepen

Responsive method, grid level

You can create a responsive grid by setting the sizing directly at the grid level. For example, given the breakpoints s, m, l, we could create a grid with full width columns on mobile, 2 columns on small screen and 4 columns on tablet with gy="g-1 gs-2 gm-4".

Source Codepen

Responsive method, columns level

You can also create a responsive grid by setting the sizing directly at the column level. For example we could create a grid with full width columns on mobile and 2 columns on small screen with gy="c-12 cs-6".

Source Codepen

Responsive method, mix grid and columns level

You can also mix the two! For example you could set the sizing at both the grid and the column level, and the grid level settings will be the fallback if a sizing is not set at the column level.

No more rows!

The biggest difference with other grid systems is that you don't need rows anymore!
